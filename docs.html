Getting Started

Fortis requires secure boot and a modern CPU with virtualization extensions. Flash the ISO to a USB drive using a verified tool and boot your system.

System Architecture

The architecture separates the hardware layer from user applications using advanced virtualization. Your physical devices are protected from malicious software by default.

Network Configuration

Networking is handled exclusively through isolated proxy VMs. Direct connections to the physical network interface are blocked at the hypervisor level.
